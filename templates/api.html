API Documentation

This API provides access to a variety of substitution ciphers. All endpoints support both GET and POST requests.

Base Endpoint

The base URL for all cipher operations is:

/api/<cipher_name>

Replace <cipher_name> with one of the supported ciphers listed below.

Request Parameters

For GET requests, pass parameters in the query string. For POST requests, send a JSON body.

Parameter Type Required? Description
text String Yes The input text to be encrypted or decrypted.
action String No The operation to perform. Can be encrypt or decrypt. Defaults to encrypt.
keyword String For mixed_alphabet The keyword required for the Mixed Alphabet cipher.
shift Integer For shift The number of positions to shift for the Shift cipher.
key String For simple_substitution A 26-character string for the Simple Substitution cipher key. If omitted, a random key is generated.

Supported Ciphers & Examples

Click on the links to see a live example.

  • atbash
    /api/atbash?text=hello
  • baconian
    /api/baconian?text=hello
  • caesar
    /api/caesar?text=hello
  • rot13
    /api/rot13?text=hello
  • polybius_square
    /api/polybius_square?text=hello
  • simple_substitution (with random key)
    /api/simple_substitution?text=hello
  • shift (requires shift)
    /api/shift?text=hello&shift=5
  • mixed_alphabet (requires keyword)
    /api/mixed_alphabet?text=hello&keyword=wednesday
